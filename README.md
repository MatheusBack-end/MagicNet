# RaknetServerLua

# Setup

```
git clone https://github.com/iMatthhzCode/RaknetServerLua
```

```
pkg install lua51;
pkg install luarocks;
pkg install libyaml;
pkg install clang;
```

```
luarocks install luasocket;
luarocks install struct;
luarocks install lyaml;
```

```
cd RaknetServerLua;
lua main.lua
```
